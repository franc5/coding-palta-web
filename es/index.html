Quines somos?

Somos un equipo de desarrolladores de software profesionales con ms de seis aos de experiencia en desarrollo web y mvil.

Proveemos soluciones de desarrollo de software para startups que buscan mejorar sus productos usando las ltimas tecnologas y las mejores prcticas de la industria.

Nuestro trabajo se apolla en el uso de metodologas giles de desarrollo como Scrum, siempre de acuerdo con las necesidades de cada cliente y proyecto.

Tecnologas

En CodingPalta usamos las mejores tecnologas para proveer servicios de desarrollo de software de alta calidad.
